# 可变宏定义

## 宏定义

C 和 C++ 中的宏定义是预处理指令，它们在编译代码之前由预处理器处理，通过预处理器**执行文本替换**。宏定义通常用于定义常量值、创建函数宏、条件编译等。主要通过 `#define` 指令实现。

- **定义常量**：在C/C++中，宏常用于定义常量。这可以使代码更易于维护，易读：

  ```C++
  #define PI 3.14159
  ```

- **创建函数宏**：也可以用来定义类似于函数的结构：

  ```
  #define SQUARE(x) ((x) * (x))
  ```

- **条件编译**：宏还常用于条件编译。这允许在编译时根据不同的宏定义包含或排除代码块。

  ```c++
  #define DEBUG
  
  #ifdef DEBUG
      printf("Debugging is enabled.\n");
  #endif
  ```

  

## 可变宏定义

可变宏定义是宏定义的一种进阶用法，在设计需要处理不定数量参数的宏时非常有用，如日志记录、错误消息生成等场景。

### 基本概念

在定义一个可变宏时，使用省略号 `...` 来表示宏可以接受一个或多个参数。在宏的替换部分，`__VA_ARGS__` 被用于引用这些可变参数。

```C++
#define DEBUG_LOG(...) printf(__VA_ARGS__)															//定义
DEBUG_LOG("Error: %s occurred in %s at line %d", error_message, __FILE__, __LINE__);				//使用
```
